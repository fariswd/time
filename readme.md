# Fancy Socket
Socket.io sending 'time' channel every seconds. With expo we handle the client. :D

## Preview
<img src="https://raw.githubusercontent.com/fariswd/time/master/ss.gif" height="300">

## Run?
```
cd server
npm install
node app.js

cd client
yarn install
expo start
```
